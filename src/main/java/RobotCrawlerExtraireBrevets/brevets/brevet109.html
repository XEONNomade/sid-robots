Titre: (FR) Système de réseau domotique et une méthode pour effectuer des transferts continus de données dans le réseau domotique
(EN) Home network system and a method for performing continuous data transfer in the home network system
(DE) Heimnetzwerksystem und ein Verfahren zur Durchführung von kontinuierlichen Datenübermittlungen in dem Heimnetzwerk
Abrégé:
(EN) A home network system for supporting continuity of data transfer destined to a mobile station (MS1) when the mobile station moves between wireless nodes (12,13), comprising a server (14) for storing multimedia data and a network (11) connecting the server to the mobile station (MS1) via the wireless nodes. A first wireless node (12), while transferring predetermined data from the server to the mobile station (MS1), senses that a connection with the mobile station is cut off. The predetermined data to be transferred to the mobile station is then buffered in one or more of the wireless nodes (13,...) at a time point of disconnection with the mobile station, and the first wireless node (12) maintains a connection with the server (14). A second wireless node (13) in the home network system establishes a connection with the mobile station (now MS1') and, in response a data transfer request from the mobile station, provides the buffered data to the MS.
