Titre (EN) HOME AUTOMATION SYSTEM
(FR) SYSTEME DE DOMOTIQUE
Abrégé:
(EN) An improved home automation system (10) that permits the user of the system (10) to remotely control the operation of various devices (30a, 30b and 30c). The system (10) is packaged in a manner that 'walks' the user through the initialization process to thereby reduce or eliminate errors associated with the initialization and set-up of the system (10). Furthermore, the configuration permits the user to initialize the system (10) without the use of tools, such as screwdrivers. The system (10) is configured with improved functionality that permits the user to readily control individual devices and/or selected groups of devices (30a, 30b and 30c)with a powerful hand-held remote transmitter (20).
(FR) L'invention concerne un système de domotique amélioré qui permet à son utilisateur de commander à distance le fonctionnement de différents dispositifs. Le système est présenté de manière à 'accompagner' l'utilisateur à travers le processus d'initialisation, afin de réduire ou d'éliminer les erreurs associées à l'initialisation et à la mise au point. En outre, la configuration de ce système permet à l'utilisateur d'initialiser le système sans recourir à des outils (par exemple, tournevis). Enfin, le système est doté de fonctionnalités améliorées qui permettent à l'utilisateur de commander aisément des dispositifs individuels et/ou des groupes de dispositifs spécifiques, au moyen d'une puissante unité de télétransmission portative.
