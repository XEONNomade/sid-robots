Titre (EN) HOME AUTOMATION EQUIPMENT COMPRISING A UNIFIED MAN/MACHINE INTERFACE
(FR) EQUIPEMENT DOMOTIQUE À INTERFACE RHM UNIFIEE
Abrégé:
(EN) The invention relates to home automation equipment comprising a unified man/machine interface. The invention consists of a central control unit (1) comprising user interface management circuits (3) which, under the control of unified user interface circuits (10), are used to: (i) select one means from among a plurality of emulation means (7, 8), corresponding respectively to one of a plurality of circuits for controlling heterogeneous players; and (ii) retransmit, to the central control unit, commands from a unified set of commands available at the unified user interface circuits (10), such that they are converted into a set of user commands specific to the selected player (29, 39).
(FR) L'équipement domotique comporte un gestionnaire central (1) comprenant des circuits de gestion d'interface utilisateur (3) pour, sous la commande de circuits d'interface utilisateur unifiée (10), sélectionner l'un, au choix, parmi une pluralité de moyens d'émulation (7, 8) de respectivement une pluralité des circuits de commande de lecteurs hétérogènes, et lui retransmettre des commandes d'un jeu unifié de commandes disponible au niveau des circuits d'interface utilisateur, unifiée (10), afin qu'elles soient converties en un jeu de commandes utilisateur spécifique au lecteur (29, 39) sélectionné.
