Titre (EN) METHOD FOR CONFIGURING A HOME AUTOMATION INSTALLATION AND TOOL FOR IMPLEMENTING SAME
(FR) PROCEDE DE CONFIGURATION D'UNE INSTALLATION DOMOTIQUE ET OUTIL POUR LA MISE EN OEUVRE D'UN TEL PROCEDE
Abrégé:
(EN) The invention relates to a method for configuring equipment in a home automation installation, each equipment being identified using an identifier stored in a memory and duplicated on an accessible and visible medium, wherein said configuration method uses a configuration tool that comprises input means, signal transmission means and at least a first and a second memory, and said method comprises an iteration of the following steps: generating, at the configuration tool, a message including the content of the first memory for receiving, after an input, the equipment identifier, and the content of the second memory for receiving, after an input, information of a first type; transmitting, via the radio-electric signal transmission means, the message in the form of an electromagnetic signal; and carrying out, at the equipment, an action that is at least partially determined by the information of the first type.
(FR) Procédé de configuration d'équipements dans une installation domotique, chaque équipement étant identifié grâce à un identifiant enregistré en mémoire et dupliqué sur un support accessible et visible, le procédé de configuration utilisant un outil de configuration comprenant un moyen de saisie, un moyen d'émission de signaux et au moins une première et une deuxième mémoire et le procédé comprenant plusieurs itérations des étapes suivantes : - générer, au niveau de l'outil de configuration, un message comprenant le contenu de la première mémoire destinée à recevoir, suite à une saisie, l'identifiant d'un équipement et le contenu de la deuxième mémoire destinée à recevoir, suite à une saisie, une information d'un premier type, - émettre, grâce au moyen d'émission de signaux radioélectriques, le message sous forme d'un signal électromagnétique, réaliser, au niveau de l'équipement, une action au moins partiellement déterminée par l'information du premier type.
