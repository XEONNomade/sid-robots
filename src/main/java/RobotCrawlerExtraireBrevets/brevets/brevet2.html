Titre (EN) COMMUNICATION SYSTEM FOR HOME AUTOMATION USING ADVANCED ADSL
(FR) SYSTEME DE COMMUNICATION POUR DOMOTIQUE UTILISANT L'ADSL AVANCEE
Abrégé:
(EN) Communication system for home automation using advanced asymmetric digital subscriber line (ADSL) is provided. Communication system for home automation using advanced ADSL includes: a home automation communication server means for providing home automation service; a home automation service channel means for interchanging data through wire or wireless line of baseband signal with object apparatus of home automation, the home automation service channel means included in ADSL terminal; and a home automation service multiplexing means for connecting the home automation communication server means and home automation service channel means through baseband signal, the home automation service multiplexing means included in ADSL connection apparatus.
(FR) L'invention concerne un système de communication pour domotique utilisant la ligne d'abonné numérique asymétrique (ADSL) avancée. Ce système de communication pour domotique utilisant l'ADSL avancée comprend : un moyen serveur de communication domotique destiné à fournir un service de domotique, un moyen voie de service de domotique permettant l'échange de données par ligne avec ou sans fil de signal en bande de base avec un appareil objet de domotique, lequel moyen voie de service de domotique est intégré dans le terminal ADSL, ainsi qu'un moyen de multiplexage de service de domotique permettant de relier le moyen serveur de communication domotique et le moyen voie de service de domotique par l'intermédiaire d'un signal en bande de base, lequel moyen de multiplexage de service de domotique est intégré dans l'appareil de connexion ADSL.
