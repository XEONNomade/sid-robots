Titre (EN) CONTROL AND COMMUNICATION PLATFORM FOR A DOMOTIC SYSTEM
(ES) PLATAFORMA DE CONTROL Y DE COMUNICACIÓN DE UN SISTEMA DOMÓTICO
(FR) PLATE-FORME DE COMMANDE ET DE COMMUNICATION D'UN SYSTÈME DOMOTIQUE
Abrégé:
(EN) The invention relates to governing hydraulic units of the hydraulic installations of each floor of a building, and touch screens for showing the data relevant to the operation of the hydraulic installation for each one-floor domicile. It is characterized in that each hydraulic unit is connected to a hydraulic unit control module (1), which, by means of an Ethernet network, is connected to the rest of the hydraulic unit control modules (1) for each floor and to a control module for the boiler room (9), which in turn includes a mobile communications antenna (11) for communication with an external network, such that it allows the hydraulic units to be governed locally or remotely, under a configuration of the control modules (1 and 9) optimized to facilitate connection of the internal elements thereof, as well as with the hydraulic units.
(ES) Comprende unidades hidráulicas de gobierno de las instalaciones hidráulicas de cada planta de una edificación, y pantallas táctiles de visualización de los datos referentes al funcionamiento de la instalación hidráulica de cada vivienda de una planta. Se caracteriza por que cada unidad hidráulica está conectada a un módulo de control de unidad hidráulica (1), que mediante una red Ethernet se conecta con el resto de módulos de control de unidades hidráulicas (1) de cada planta y con un módulo de control de sala de calderas (9) que a su vez comprende una antena de comunicaciones móvil (11) para comunicación con una red externa, de forma que se permite gobernar las unidades hidráulicas, local o remotamente, y todo ello con una configuración de los módulos de control (1 y 9) optimizada para facilitar el conexionado de sus elementos internos que los componen, así como con las unidades hidráulicas.
(FR) La présente invention concerne une plate-forme de commande et de communication d'un système domotique comprenant des unités hydrauliques de commande des installations hydrauliques de chaque étage d'un bâtiment, ainsi que des écrans tactiles de visualisation des données concernant le fonctionnement de l'installation hydraulique de chaque logement d'un étage. L'invention est caractérisée en ce que chaque unité hydraulique est connectée à un module de commande d'unité hydraulique (1) qui, au moyen d'un réseau Ethernet, se connecte au reste des modules de commande d'unité hydraulique (1) de chaque étage et à un module de commande de chaufferie (9) qui comprend à son tour une antenne de communication mobile (11) permettant de communiquer avec un réseau externe, de manière qu'il soit possible de commander les unités hydrauliques de façon locale ou à distance, avec une configuration des modules de commande (1 et 9) optimisée pour faciliter la connexion de leurs éléments internes constitutifs et avec les unités hydrauliques.
