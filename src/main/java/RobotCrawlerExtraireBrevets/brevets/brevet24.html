Titre (EN) SYSTEM AND METHOD FOR HOME AUTOMATION USING WIRELESS CONTROL RF REMOCON MODULE BASED ON NETWORK
(FR) SYSTEME ET PROCEDE DE DOMOTIQUE FAISANT APPEL A UN MODULE DE TELECOMMANDE RF SANS FIL BASE SUR UN RESEAU
Abrégé:
(EN) A home automation system using an infrared (IR)/radio frequency (RF) integration remote control module based on a network. At least one IR/RF integration remote control module (100) is disposed at a predetermined location of a home automation region. The IR/RF integration remote control module (100) with a wireless interface function integrates an RF modulation/reception/transmission function and an IR modulation/reception/transmission. A set-top box (200) is appropriately disposed so that it can carry out an interface function for controlling the IR/RF integration remote control module (100) and a wired/wireless Internet access function. The set-top box (200) receives various control commands over an Internet network using transmission control protocol/Internet protocol (TCP/IP) and controls the IR/RF integration remote control module (100). Even though various pieces of equipment using the conventional IR reception/transmission scheme or a new electronic product equipped with an RF module is added to the home automation system using an RF modulation scheme, an automation operation and a bi­directional control operation based. on the integrated IR and RF transmission scheme can be appropriately carried out. Thus, the home automation system and method can be cost-effectively and conveniently implemented.
(FR) L'invention concerne un système de domotique comportant un module de télécommande d'intégration à infrarouge (IR)/fréquence radio (RF), basé sur un réseau. Au moins un module de télécommande d'intégration IR/RF (100) a un emplacement déterminé dans une zone de domotique. Ce module de télécommande d'intégration IR/RF (100), doté d'une fonction interface sans fil, intègre une fonction RF de modulation/réception/transmission et une fonction IR de modulation/réception/ transmission. Un décodeur (200) est placé de façon adéquate pour exécuter une fonction interface de commande du module de télécommande d'intégration IR/RF (100) et une fonction d'accès Internet avec/sans fil. Ce décodeur (200) reçoit plusieurs ordres de contrôle par un réseau Internet utilisant le protocole de contrôle de transmission/protocole Internet (TCP/IP) et il commande le module de télécommande d'intégration IR/RF (100). Même lorsque plusieurs équipements fonctionnant selon le schéma de réception/ transmission IR classique ou un nouvel appareil électronique pourvu d'un module RF sont ajoutés au système domotique fonctionnant selon le schéma de modulation RF, une opération d'automation et une opération de commande bidirectionnelle peuvent être exécutées de manière adéquate sur la base du schéma de transmission intégré IR et RF. Ainsi, le système et le procédé de domotique de la présente invention peuvent être appliqués au mieux en termes de confort et de rentabilité.
