Titre: (FR) Procédé d'utilisation en réseau d'un processeur de logique floue, et processeur de logique floue utilisable en réseau
(EN) Method of using fuzzy logic processor in a network and fuzzy logic processor usable in a network
(DE) Verfahren zur Benutzung eines Fuzzy-Logic-Prozessors in einem Netzwerk und dafür geeigneter Fuzzy-Logic-Prozessor
Abrégé:
(FR) Pour permettre la mise en place d'un processeur (1) flou dans un réseau (R) de communication domotique on prévoit que ce processeur flou sera capable de gérer les messages transitant sur le réseau. Dans ce but, ce processeur flou fonctionne en deux temps. Dans un premier temps (1), sur une première partie du message, le processeur flou détermine si le message lui est adressé en fonction de règles lui permettant de savoir s'il est le destinataire. Une fois qu'il sait qu'il est le destinataire, il sait aussi, parce que l'adresse de l'émetteur du signal est présente dans le message, produire un ordre d'aiguillage (8). Cet ordre d'aiguillage est appliqué sur un multiplexeur (14) situé à l'entrée en interface entre le processeur flou et le réseau pour orienter des données, une autre partie du message, vers l'entrée (5) adéquate de ce processeur flou.
(EN) The method involves the processor (2) receiving messages from the network (R) through a multiplexer (14) and circuits (9) which read batches of correspondence tables and membership functions. Memories (3,4) store processing rules and membership functions and the initial output (8) from the processor produces a switching order to the multiplexer to guide data in a second part of the message to the appropriate entry (5). The fuzzy processor therefore functions in two stages. At first data is extracted from a message then it is introduced into the appropriate processor entry for fuzzy logic processing.
