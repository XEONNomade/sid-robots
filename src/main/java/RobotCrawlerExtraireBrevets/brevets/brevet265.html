Titre: (FR) Système domestique pour le contrôle intelligent de l'habitat
(EN) Household system for the intelligent control of the living environment
(DE) Haushaltssystem zur intelligenten Steuerung des Wohnbereichs
Abrégé:
(FR) Le système domotique (10) se compose d'un câble ou bus (15) qui devra parcourir toute l'installation, grâce à quoi il forme d'abord un système ouvert. Sur le câble ou bus (15) sont greffés, d'un côté, des modules spéciaux (12), (13) et (14), qui représentent les différentes possibilités du système opérationnel, soit à travers les ordres reçus d'un ordinateur (11) en programmant le module (13), soit par d'autres systèmes différents, non dirigés par un ordinateur (11) et qui auraient comme base de fonctionnement des signaux de type infrarouge, comme le module (12) ou bien un autre type de signal, qui pourrait être celui qui serait représenté dans un module (14) par une base de temps ou à partir de signaux plus ou moins importants d'une certaine fréquence.
(EN) Has intelligent nodes which do not require central control. The system is of the type which uses a bus topology for interconnection of the various controls and appliances. The system (10) is configured from a bus or cable (15), which runs around the whole installation and is of the distributed system type with communication control to which are associated special modules (12, 13,14). These modules (12,13,14) are used to program nodes (18,19) which a connected directly and independently with respect to others on the bus (15). The system (10) is fed internally by 24 to 28 volt direct currents and externally that is with respect to appliances to be installed, at 110 to 220 volts alternating current. The inputs and outputs of all the system are connected independently to the nodes (18,19).
