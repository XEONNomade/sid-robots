Titre (EN) METHOD AND APPARATUS FOR PROVIDING DISTRIBUTED CONTROL OF A HOME AUTOMATION SYSTEM
(FR) PROCEDE ET APPAREIL DESTINES A FOURNIE UNE COMMANDE REPARTIE D'UNE INSTALLATION DOMOTIQUE
Abrégé:
(EN) A method and apparatus for providing distributed control of a home automation system (2) is provided. Each device (6, 8, 10, 12, 14) participating in a home automation system is equipped with control logic providing distributed control. Through the control logic, each device maintains scene definitions describing the state of the device for each scene in which it participates. When any device in the system receives a request to launch a scene, such as a button press on one of the devices, the device broadcasts a scene state change message to all devices within the home automation system (2). The scene state change message identifies to each device a particular scene that should be launched. Each device in the system receives the message and determines whether the device is a participant in the scene. If the device is a participant in the scene, the device adjusts its state according to a scene definition stored in the device associated with the scene. The device may adjust a controlled electrical load, such as a light, according to the scene definition by turning the load on, off, or setting the load to some intermediate value.
(FR) L'invention concerne un procédé et un appareil destinés à fournir une commande répartie d'une installation domotique (2). Chaque dispositif (6, 8, 10, 12, 14) faisant partie de l'installation domotique est équipé d'une logique de commande fournissant la commande répartie. Grâce à la logique de commande, chaque dispositif maintient des définitions de la scène décrivant l'état du dispositif pour chaque scène à laquelle il fait partie. Lorsqu'un dispositif quelconque de l'installation reçoit une demande de lancement d'une scène, tel qu'une pression de bouton d'un des dispositifs, le dispositif diffuse un message de changement d'état d'une scène à tous les dispositifs de l'installation domotique (2). Le message de changement d'état d'une scène identifie pour chaque dispositif une scène particulière qui devrait être lancée. Chaque dispositif de l'installation reçoit le message et détermine si le dispositif fait partie de la scène. Si oui, le dispositif ajuste son état selon une définition de la scène stockée dans le dispositif associé à la scène. Le dispositif peut ajuster une charge électrique commandée, telle que l'éclairage, selon la définition de la scène en mettant en marche, en arrêtant la charge ou en la réglant à une valeur intermédiaire.
