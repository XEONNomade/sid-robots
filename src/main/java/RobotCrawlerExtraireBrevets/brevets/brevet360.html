(EN) Analog clock for household appliance, in particular for oven.
(DE) Zeigernuhr für ein Hausgerät, insbesondere für einen Backofen
Abrégé:
(EN) A clock (1) for household appliance is disclosed, comprising: hands (L1, L2) to tell the time, a stepping motor (M) to actuate said hands, icons (I1-I5) to indicate clock functions, a switch (SW) to control the switch-on/off of the appliance according to a time period set by the user, a microcontroller (MC) connected to the stepping motor (M), icons (I1-I5) and switch (SW) to control the operation of these devices, buttons (T1, T2, T3) connected to the microcontroller (MC) and actuated by the user to select and control the clock functions, and a power supply (A) connected to the mains to power said microcontroller (MC).
