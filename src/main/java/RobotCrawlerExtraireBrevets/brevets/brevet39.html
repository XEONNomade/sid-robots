Titre (EN) SWITCHING VALUE CONTROLLER, INTERACTIVE TERMINAL AND INTELLIGENT HOME CONTROL SYSTEM AND METHOD
(FR) CONTRÔLEUR DE VALEUR COMMUTÉE, TERMINAL INTERACTIF AINSI QUE SYSTÈME ET PROCÉDÉ DOMOTIQUE INTELLIGENT
(ZH) ?????????????????????????
Abrégé:
(EN) A switching value controller (2) comprises a power line communication module (10), a micro control module (20) and a switching value output module (30). The power line communication module (10) is used for extracting power line communication signals, demodulating the power line communication signals and converting the power line communication signals into digital signals. The micro control module (20) is used for analyzing whether each digital bit representing switching value state in the digital signals is a first digit, and outputting high level if it is yes or outputting low level if it is not. The switching value output module (30) is used for amplifying signal power and driving electrical appliances at load ends (31~3n). An interactive terminal (1) is used for receiving home-mode selection instructions, generating switch value control data, modulating the switch value control data into the power line communication signals and sending the power line communication signals. An intelligent home control system and a method thereof are provided. The system comprises the switching value controller (2) and the interactive terminal (1). The devices, system and the method can greatly reduce the cost of the intelligent home control system and improve the control efficiency of the intelligence home control system.
(FR) L'invention concerne un contrôleur de valeur commutée (2) comprenant un module de communication sur ligne d'alimentation (10), un module microcontrôleur (20) et un module de sortie de valeur commutée (30). Le module de communication sur ligne d'alimentation (10) est utilisé pour extraire des signaux de communication de la ligne d'alimentation, démoduler les signaux de communication de la ligne d'alimentation et convertir les signaux de communication de la ligne d'alimentation en signaux numériques. Le module microcontrôleur (20) est utilisé pour analyser si chaque bit numérique représentant l'état de valeur commutée dans les signaux numériques est un premier digit et pour délivrer en sortie un niveau logique haut si la réponse est oui et un niveau logique bas si la réponse est non. Le module de sortie de valeur commutée (30) est utilisé pour amplifier la puissance du signal et commander des appareils électriques du côté des charges (31~3n). Un terminal interactif (1) est utilisé pour recevoir des instructions de sélection du mode domestique, générer des données de commande de valeur commutée, moduler les données de commande de valeur commutée sur les signaux de communication sur ligne d'alimentation et envoyer les signaux de communication sur ligne d'alimentation. L'invention concerne un système domotique intelligent et un procédé à cet effet. Le système comprend le contrôleur de valeur commutée (2) et le terminal interactif (1). Les dispositifs, le système et le procédé peuvent nettement réduire le coût du système domotique intelligent et améliorer l'efficacité de commande du système domotique intelligent.
