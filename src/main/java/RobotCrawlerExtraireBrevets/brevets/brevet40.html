Titre (EN) DEVICE, SYSTEM AND METHOD FOR IMPLEMENTING SMART HOME APPLICATIONS
(FR) DISPOSITIF, SYSTÈME ET PROCÉDÉ DE MISE EN ŒUVRE D'APPLICATIONS DE MAISON INTELLIGENTE
(ZH) ???????????????????
Abrégé:
(EN) A device for implementing smart home applications is disclosed in the present invention, and said device includes a smart home gateway, a home control center and a terminal proxy device. The smart home gateway is used to perform the message transformation between the home control center and the terminal proxy device and send the transformed message to the home control center or the terminal proxy device; the home control center is used to perform smart home control logic according to user instructions or a pre-set task, send an instruction to the local terminal device via the terminal proxy device through the smart home gateway, and implement smart home service through the instruction implementation of the local terminal device; the terminal proxy device is used to perform the protocol transformation between the smart home gateway and the local terminal device. A system and method for implementing smart home applications are also disclosed in the present invention. The present invention enables the smart home applications based on the uniform control on terminal devices.
(FR) La présente invention porte sur un dispositif de mise en œuvre d'applications de maison intelligente, ledit dispositif comprenant une passerelle de maison intelligente, un centre domotique et un dispositif mandataire de terminal. La passerelle de maison intelligente est utilisée pour effectuer la transformation de message entre le centre domotique et le dispositif mandataire de terminal et pour transmettre le message transformé au centre domotique ou au dispositif mandataire de terminal ; le centre domotique est utilisé pour exécuter une logique de commande de maison intelligente conformément à des instructions d'utilisateur ou à une tâche prédéfinie, transmettre une instruction au dispositif terminal local par l'intermédiaire du dispositif mandataire de terminal en passant par la passerelle de maison intelligente, et mettre en œuvre un service de maison intelligente par l'exécution de l'instruction du dispositif terminal local ; le dispositif mandataire de terminal est utilisé pour effectuer la transformation de protocole entre la passerelle de maison intelligente et le dispositif terminal local. La présente invention porte également sur un système et un procédé de mise en œuvre d'applications de maison intelligente. La présente invention permet les applications de maison intelligente sur la base de la commande uniforme sur des dispositifs terminaux.
