Titre (EN) SYSTEM FOR CONTROLLING DOSMETIC APPLIANCES AND METHOD FOR IMPLEMENTING A CONTROL SYSTEM
(FR) SYSTEME DE COMMANDE D'EQUIPEMENTS DOMOTIQUES ET PROCEDE DE MISE EN OEUVRE D'UN SYSTEME DE COMMANDE
Abrégé:
(EN) The present invention relates to a system (10) for controlling at least one domestic appliance (21-27) fitted in a building (1), the system (10) comprising a control point (30) controlling the one or more domestic appliances (21-27) fitted in the building (1); a gateway (40) for communicating between the control point (30) and at least one mobile terminal (50) belonging to a user (U50) of the building (1); and at least one primary marker (81-84) that can be read by hardware and software recognition means of the mobile terminal (50), the primary marker (81-84) being associated with at least one preset domestic appliance (21-24). The system (10) is characterized in that the control point (30) either operates in a normal mode, the control point (30) independently controlling the domestic appliance, or operates in an advanced mode, the control point (30) controlling the domestic appliance (21-27) depending on information transmitted by the mobile terminal (50) communicating with the control point (30) via the gateway (40); and in that identification of the primary marker (81-84), by the identifying means (32) of the control point (30), permits the advanced mode to be activated so that the one or more domestic appliances (21-27) associated with this primary marker (81-84) can be controlled.
(FR) La présente invention concerne un système (10) de commande d'au moins un équipement domotique (21 -27) équipant un bâtiment (1), le système (10) comprenant un point de commande (30) pilotant le ou les équipements domotiques (21-27) équipant le bâtiment (1); une passerelle (40) de communication entre le point de commande (30) et au moins un terminal nomade (50) appartenant à un usager (U50) du bâtiment (1); et au moins un marqueur primaire (81-84) lisible par des moyens matériels et logiciels de reconnaissance équipant le terminal nomade (50), le marqueur primaire (81-84) étant associé à au moins un équipement domotique (21-24) prédéterminé. Le système (10) est caractérisé en ce que le point de commande (30) fonctionne soit dans un mode normal, où le point de commande (30) pilote indépendamment l'équipement domotique (21-27), soit dans un mode avancé, où le point de commande (30) pilote l'équipement domotique (21-27) en fonction d'informations transmises par le terminal nomade (50) en communication avec le point de commande (30) par l'intermédiaire de la passerelle (40); et en ce que l'identification du marqueur primaire (81-84), par des moyens d'identification (32) équipant le point de commande (30), autorise une activation du mode avancé pour le pilotage du ou des équipements domotiques (21-27) associés à ce marqueur primaire (81-84).
