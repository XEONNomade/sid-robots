Titre: (FR) Dispositif de commande d'appareils de climatisation
(EN) Device for controlling air-conditioning devices
(DE) Vorrichtung zur Steuerung von Klimaanlagen
Abrégé:
(FR) Il est décrit un dispositif de commande d'appareils de climatisation intégré à un système domotique. Ce dispositif dispose d'un jeu universel de commandes d'appareils de climatisation. Il est donc pilotable par le système domotique à l'aide de ce jeu universel de commandes. D'autre part, ce dispositif dispose de moyens permettant de recevoir et d'enregistrer les séquences infrarouges émises par une télécommande d'appareils de climatisation et de les associer avec les commandes du jeu de commandes universel. Il dispose également des moyens d'émettre la séquence infrarouge associée à une commande du jeu universel de commandes lorsque cette commande est sollicitée par le système domotique. Le dispositif de commande est donc à même d'apprendre le jeu de commandes d'un appareil de climatisation particulier.
(EN) The device (1.2) has an interpretation unit for interpreting a set of universal controls of an air conditioner. A control unit controls an air conditioning apparatus (1.3), and includes a set of controls. A recording unit i.e. memory, stores an infra red sequence corresponding to a control among the set of controls of the air conditioning apparatus. An infrared emitter emits the infra-red sequence to the apparatus when the emitter receives the control among the set of universal controls sent by a home automation system.
