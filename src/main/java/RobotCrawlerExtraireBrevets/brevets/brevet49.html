Titre: (FR) Capteur vent avec seuils par produits
(EN) Wind sensor with thresholds by products
(DE) Windsensor mit Schwellenwerten je nach Produkt
Abrégé:
(FR) Capteur de sécurité au vent, comprenant un dispositif de mesure physique d'au moins un paramètre lié à l'action du vent, comprenant un moyen bidirectionnel de communication radiofréquences, apte à émettre un signal vers au moins un élément domotique mobile lorsque le paramètre dépasse un seuil, comprenant une table d'affectation d'au moins un premier seuil et un deuxième seuil relatifs à des types distincts d'éléments domotiques mobiles et tel que le signal émis comprend un identifiant du type dont le seuil a été dépassé, un code de commande d'une action de sécurité et/ou un identifiant du capteur de sécurité. Installation domotique comprenant un tel capteur et dispositif de commande d'une telle installation domotique.
(EN) The wind safety sensor (24) has a physical device for measuring at least one parameter associated with the action of the wind. A radio-frequency communication unit transmits a signal when the parameter exceeds a threshold. An assignment table (41) stores at least one first threshold and one second threshold relating to distinct types of mobile home automation elements. The transmitted signal includes an identifier of a type for which the threshold has been exceeded, and an information selected from: a code controlling a safety action; an identifier of the safety sensor; or their combinations. Independent claims are also included for the following: (1) a home automation installation; and (2) a control device for a home automation installation.
