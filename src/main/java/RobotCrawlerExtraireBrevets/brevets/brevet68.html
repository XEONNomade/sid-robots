Titre (DE) TERMINAL ZUR BEDIENUNG EINER HAUSTECHNIKANLAGE
(EN) TERMINAL FOR OPERATING A HOME AUTOMATION INSTALLATION
(FR) TERMINAL DE COMMANDE D'UNE INSTALLATION DOMOTIQUE
Abrégé:
(DE) Terminal mit einer Tastatur, die zur Bedienung einer Haustechnikanlage geeignet ist. Das Terminal ist als Mobilgerät ausgebildet und weist einen drahtlosen Sendeempfänger auf, der mit der Basisstation eines schnurlosen Telefons mit der gleichen Übertragungstechnik Daten für die Haustechnikanlage austauschen kann, mit der im Telefonbetrieb die Sprache übertragen wird. Zusätzlich ist das Terminal für den normalen Telefonverkehr als Mobilstation der Basisstation des schnurlosen Telefons ausgebildet, oder es ist als Akustikkoppler zum Austausch der Daten über den Handapparat eines Fernsprechapparates ausgebildet.
(EN) A terminal has a keyboard suitable for operating a home automation installation. The terminal is designed as a mobile apparatus and has a wireless sender-receiver which can exchange data for the home automation installation with the basis station of a wireless telephone by the same transmission technique used for speech-transmission by telephone. The terminal is also designed for normal telephone communication as the mobile station of the wireless telephone basis station, or it is designed as an acoustic coupler to exchange data through the handset of a telephone apparatus.
(FR) Le terminal décrit comprend un clavier propre à commander une installation domotique. Le terminal est constitué d'un appareil mobile et comprend un récepteur-émetteur sans fil qui peut échanger des données de commande de l'installation domotique avec la station de base d'un téléphone sans fil par la même technique de transmission utilisée pour transmettre la parole par téléphone. Le terminal sert également de station mobile associée à la station de base du téléphone sans fil et assure ainsi les communications téléphoniques normales ou forme un coupleur acoustique d'échange de données par l'intermédiaire du combiné d'un appareil téléphonique.
