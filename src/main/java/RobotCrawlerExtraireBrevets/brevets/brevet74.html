(FR) DISPOSITIF DE MÉMORISATION DE DONNÉES, SERVEUR DE GESTION, CIRCUIT INTÉGRÉ, SYSTÈME D'ACTUALISATION DE DONNÉES, APPAREILS DE DOMOTIQUE, PROCÉDÉ D'ACTUALISATION DE DONNÉES, PROCÉDÉ DE CRYPTAGE, PROCÉDÉ DE GÉNÉRATION DE CLÉ
(JA) ????????????????????????????????????????????????????????????
Abrégé:
(EN) Provided is a data storage device capable of safely and effectively updating software of a home electric apparatus. In the home electric apparatus (100) in which currently used data is to be updated to new data, a reception unit (140) receives encrypted new data and a serial number of data to be updated. A key generation unit (112) generates a key by executing a predetermined irreversible calculation on the unique information correlated to the currently used data by a number of times based on the serial number of the data to be updated. An update unit (113) decrypts the new data by using the key.
(FR) L'invention concerne un dispositif de mémorisation de données apte à actualiser de manière sûre et efficace le logiciel d'un appareil de domotique. Dans cet appareil de domotique (100), dont une donnée utilisée doit être actualisée et remplacée par une nouvelle donnée, une unité de réception (140) reçoit une nouvelle donnée cryptée et un numéro d'ordre de la donnée à actualiser. Une unité de génération de clé (112) génère une clé par l'exécution d'un calcul prédéfini irréversible sur la seule information en corrélation avec la donnée utilisée au moyen d'un nombre de temps basé sur le numéro d'ordre de la donnée à actualiser. Une unité d'actualisation (113) décrypte la nouvelle donnée au moyen de la clé.
