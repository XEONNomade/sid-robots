Titre (EN) SYSTEM, METHOD AND SOFTWARE FOR HOUSE AUTOMATION CONFIGURATION AT POINT OF SALE OF AN AUTOMATION DEVICE
(FR) SYSTÈME, PROCÉDÉ ET LOGICIEL POUR LA CONFIGURATION DOMOTIQUE AU NIVEAU D'UN POINT DE VENTE D'UN DISPOSITIF D'AUTOMATISATION
Abrégé:
(EN) Point of sale device, system and computer program for configuration of house automation, including a reader (7) arranged to read an identity associated with a sensor, included in product (2), for home automation, an interface (5) arranged to receive input of installation data of the sensor; and a communication module arranged to send the installation data and associated identity of the sensor to a server (8). When the customer installs the product (2), which may include a wireless bridge (11), the wireless bridge will communicate with a gateway (18) which in turn retrieves the identity of the sensor from the server (8).
(FR) L'invention porte sur un dispositif de point de vente, un système et un programme d'ordinateur pour la configuration d'un produit domotique, comprenant un lecteur (7) conçu pour lire une identité associée à un détecteur, compris dans un produit (2), pour la domotique, une interface (5), conçue pour recevoir une entrée de données d'installation du détecteur; et un module de communication conçu pour envoyer les données d'installation et l'identité associée du détecteur à un serveur (8). Lorsque le client installe le produit (2), qui peut comprendre un pont sans fil (11), le pont sans fil communique avec une passerelle (18) qui à son tour récupère l'identité du détecteur à partir du serveur (8).
