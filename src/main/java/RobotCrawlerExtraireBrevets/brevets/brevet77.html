(EN) Regarding an information processing device forming a home network, a technology for more effectively updating a firmware is provided. Provided are a remote management means for communicating with a server device which stores software to be delivered, and a controlled device within a home network, and for managing applications; a service management program for managing services of the controlled device; a firmware management program for managing a firmware of the controlled device; a gateway firmware management program for managing a firmware of a gateway device; a gateway firmware updating program for rewriting the firmware of the gateway device; and a dependence relationship management program which, when a software is to be distributed, analyzes a dependence relationship aggregation table aggregating contents of a dependence relationship table which describes a software whereon the software to be distributed depends (refer to FIG. 1).
(FR) En ce qui concerne un dispositif de traitement d'informations formant un réseau domotique, une technologie de mise à jour plus efficace d'un micrologiciel est proposée. Le système comprend un moyen de gestion à distance pour communiquer avec un dispositif serveur qui mémorise un logiciel à délivrer et un dispositif contrôlé dans un réseau domotique, et pour gérer des applications ; un programme de gestion de service pour gérer des services du dispositif contrôlé ; un programme de gestion de micrologiciel pour gérer un micrologiciel du dispositif contrôlé ; un programme de gestion de micrologiciel de passerelle pour gérer un micrologiciel d'un dispositif passerelle ; un programme de mise à jour de micrologiciel de passerelle pour réécrire le micrologiciel du dispositif passerelle ; et un programme de gestion de relation de dépendance qui, lorsqu'un logiciel doit être distribué, analyse une table d'agrégation de relations de dépendance agrégeant les contenus d'une table de relations de dépendance qui décrit un logiciel duquel le logiciel à distribuer dépend (voir la figure 1).
