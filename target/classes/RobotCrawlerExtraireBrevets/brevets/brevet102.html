Titre: (FR) Dispositif d'interfaçage entre supports de communication dans un réseau de domotique
(EN) Interface circuit between communication media in a home network
(DE) Schnittstellenvorrichtung zwischen Haus-Netzwerkübertragungsmedien
Abrégé:
(FR) L'invention concerne un dispositif d'interfaçage (13) entre au moins deux bus de communication (11, 12). Le dispositif est caractérisé en ce qu'il comprend une interface à double sens (1, 2) pour chacun desdits bus de communication, un circuit de comparaison (5) de messages reçus sur lesdits interfaces, un circuit de résolution de conflits de priorité (6) entre des messages se chevauchant temporellement, ledit circuit de résolution de conflits (6) autorisant la transmission, à travers le ou lesdits interfaces (1,2), du message prioritaire vers les bus de communication dont ledit message prioritaire n'est pas originaire. L'invention s'applique notamment dans le domaine de la domotique.
(EN) The device includes a bidirectional interface for each communication bus which is connected to A circuit compares messages received at the interfaces. A processor recognises priority messages and resolves information conflicts. The processor authorises transmission across interfaces of priority messages towards the communication bus where the message is to be sent. The device is pref. a repeater relay which allows domestic household appliances to be controlled from a central TV, satellite and video controller.
