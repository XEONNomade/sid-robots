Abrégé:
(EN) Disclosed are a home automation system, a method for the system, and an information processing center device. The method comprises: a first terminal node sending to an information processing center a control message for a second terminal node (S10); the information processing center processing the control message and sending same to the second terminal node (S20); the second terminal node executing a corresponding operation on the basis of the control message (S30). The present invention enables communication among and management of physical device nodes in a home, and by using a message registration distribution mechanism as a replacement for a traditional node-to-node communication mechanism, decreases the coupling of physical nodes when implementing home automations, increases the expansibility and stability of home automation systems, also improves communication efficiency, and reduces communication costs.
(FR) L'invention porte sur un système domotique, un procédé pour le système et un dispositif de centre de traitement d'informations. Le procédé comprend les opérations suivantes : un premier nœud terminal envoie à un centre de traitement d'informations un message de commande destiné à un second nœud terminal (S10); le centre de traitement d'informations traite le message de commande et l'envoie au second nœud terminal (S20); le second nœud terminal exécute une opération correspondante sur la base du message de commande (S30). La présente invention permet une communication entre des nœuds de dispositif physique dans un domicile et une gestion de ces nœuds, et par utilisation d'un mécanisme de distribution d'enregistrement de message en remplacement d'un mécanisme de communication de nœud à nœud classique, réduit le couplage de nœuds physiques lors de la mise en œuvre de systèmes domotiques, augmente l'extensibilité et la stabilité de systèmes domotiques, améliore également l'efficacité de communication et réduit les coûts de communication.
