Titre (DE) VERFAHREN ZUM FERNBEDIENEN EINER DOMOTIK-ANLAGE UND MOBILES DATENENDGERÄT FÜR EINE DOMOTIK-ANLAGE
(EN) METHOD FOR REMOTELY CONTROLLING A HOME AUTOMATION SYSTEM AND A MOBILE DATA TERMINAL FOR A HOME AUTOMATION SYSTEM
(FR) PROCEDE DE TELECOMMANDE D'UNE INSTALLATION DOMOTIQUE ET TERMINAL DE DONNEES MOBILE DESTINE A CETTE INSTALLATION DOMOTIQUE
Abrégé:
(DE) Verfahren und Datenendgerät zum Fernbedienen einer Domotik-Anlage über eine dem Standort der Domotik-Anlage (1) lokal zugeordnete Schnittstelle (7), die Fernbedienungssignale von einem Datenendgerät (11) erhält, das sich ausserhalb des Standortes der Domotik-Anlage befindet, insbesondere in einem Kraftfahrzeug (10). Das Datenendgerät (11) ist mobil, und der für die Fernbedienung erforderliche Datenverkehr zwischen dem mobilen Datenendgerät (11) und der lokalen Schnittstelle (7) wird über eine mobile Schnittstelle (18) des Datenendgerätes (11) und über das Internet abgewickelt.
(EN) The invention relates to a method and data terminal for remotely controlling a home automation system via an interface (7) which is locally assigned to the location of the home automation system (1). Said interface receives the remote control signals from a data terminal (11) situated away from the location of the home automation system, especially in a motor vehicle (10). The data terminal (11) is mobile, and the data traffic, said data traffic being required for the remote control, between the mobile terminal (11) and the local interface (7) is handled via a mobile interface (18) of the data terminal (11) and via the internet.
(FR) L'invention concerne un procédé et un terminal permettant de télécommander une installation domotique par une interface (7) associée localement au lieu d'implantation de l'installation domotique (1). Cette interface reçoit les signaux de télécommande d'un terminal de données (11) qui se trouve en dehors du lieu d'implantation de l'installation domotique, notamment dans un véhicule (10). Le terminal de données (11) est mobile et le trafic de données entre le terminal mobile (11) et l'interface locale (7), nécessaire à la télécommande, se déroule par une interface mobile (18) du terminal de données (11) et via l'Internet.
