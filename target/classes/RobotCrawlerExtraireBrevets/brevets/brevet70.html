Titre (EN) METHOD FOR CONTROLLING HOME AUTOMATION SYSTEM
(FR) PROCEDE DE COMMANDE DE SYSTEME DE DOMOTIQUE
Abrégé:
(EN) A method for controlling a home automation system is disclosed, in which web sites, a PC trying to connect to the web sites, and electric home appliances connected to the PC are provided. The home automation system includes a step for trying to communicate with electric home appliances and web sites once a user operates an internet operating program; a step for opening a main web page, analyzing commands for a button clicked by a user with reference to the main web page, and performing a controlling operation according to the analyzed result if the PC is connected to the web sites; and a step for opening a local page stored in PC and opening a corresponding link page according to a button clicked by a user if the PC is connected only to electric home appliances without being connected to the web sites.
(FR) L'invention concerne un procédé de commande de système de domotique, qui fait intervenir des sites Web, un ordinateur personnel tentant d'accéder à ces sites, et des appareils électroménagers reliés à l'ordinateur personnel. Le système de domotique fonctionne selon les étapes suivantes: tentative de communication avec les appareils électroménagers et les sites Web lorsque l'utilisateur lance un programme d'exploitation Internet; ouverture d'une page Web principale, analyse des commandes correspondant au clic de l'utilisateur par rapport à la page Web principale, et commande d'intervention en fonction du résultat de l'analyse si l'ordinateur personnel est relié aux sites Web; et enfin ouverture d'une page locale enregistrée dans l'ordinateur personnel puis ouverture d'une page de lien correspondante selon le clic de l'utilisateur si l'ordinateur personnel est relié seulement aux appareils électroménagers, sans connexion aux sites Web.
